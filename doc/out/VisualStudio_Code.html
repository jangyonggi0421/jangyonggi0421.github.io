Visual Studio Code(Windows) 설치

1. 설치 파일 다운로드 : https://code.visualstudio.com/

2. 설치파일 실행 후 [동의합니다] 선택하고 [다음] 버튼 클릭

3. 설치 위치 선택하고 [다음] 버튼 클릭

4. 시작 메뉴 폴더 선택 항목은 필요에 따라 설정하고 [다음] 버튼 클릭

5. 추가 작업 선택 >> 필요에 따라 설정하고 [다음] 버튼 클릭

6. [설치] 버튼 클릭


7. [종료] 버튼 클릭

  • 한글화 적용

    1) VS Code의 Extension(Ctrl+Shift+X)을 실행하고 검색창에 korean 입력

    2) install 클릭
    3) Change Language and Restart 클릭
