- macOS Tahoe 26.0.1 버전 기준
  1. 우선 Git 설치 확인 필요
    • 터미널에 다음과 같이 입력: -MacBookPro ~ % git --version
    • macOS의 경우 기본적으로 Git이 설치되어 있음. 없을 경우 다음 URL을 통해 설치하고 진행
  2. 다운로드 URL : https://github.com/apps/desktop?locale=ko-KR
  3. Download now 클릭
  4. M1, M2 등 Apple silicon 칩일 경우 Download for Apple silicon 클릭, 이 외 intel 칩일 경우 Download for Intel chip 클릭
    • Windows의 경우 하단 Download for Windows 클릭
  5. 파일 다운로드 후 열기 버튼 클릭
  6. GitHub 계정으로 로그인(Sign in to GitHub.com)
  7. 데스크톱 인증(Authorize desktop)
  8. Git 설정
    • Use my GitHub account name and email address(내 GitHub 계정명과 이메일 주소 사용) 선택하고 이메일 주소를 입력하고 [Finish] 클릭
  9. Move and Restart
  10. 기존 생성된 Repository(저장소) 선택(없을 경우 Repository 생성)
  11. [Clone ~~~] 클릭
  12. [Clone] 또는 필요한 설정이 있을 경우 수정하고 [Clone]
  13. 설치 및 연결 완료, 테스트
