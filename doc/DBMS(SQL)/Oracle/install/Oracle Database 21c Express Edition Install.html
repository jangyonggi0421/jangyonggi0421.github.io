Oracle Database 21c Express Edition 설치 및 설정

시스템 환경 및 리소스 제한
  • 접속 OS: Windows 11 Pro (25H2, 26200.7922)
  • 데이터 크기: 최대 12GB (사용자 데이터 기준)
  • 메모리(RAM): 최대 2GB 활용
  • 프로세서: 최대 2개의 스레드 사용
  1. 파일 다운로드 및 압축해제

    Oracle Database 21c XE 다운로드 페이지

  2. setup.exe 파일 실행

    압축 해제된 OracleXE213_Win64 폴더 내에서 실행합니다.

  3. 라이선스 동의 및 다음
  4. 설치 경로 지정

    기본 경로를 사용하거나 [변경]을 통해 원하는 위치(예: H:\dev\OracleXE)를 지정합니다.

  5. 관리자 계정 비밀번호 설정

    SYS, SYSTEM, PDBADMIN 계정의 비밀번호를 설정합니다. (예: 12345)

  6. 설치 확인

    명령 프롬프트(CMD)에서 sqlplus를 입력하여 정상 작동 여부를 확인합니다.

  7. 환경 변수 추가

    고급 시스템 설정의 환경 변수 메뉴에서 다음 항목을 추가합니다.

    ORACLE_HOME C:\app\사용자명\product\21c\dbhomeXE (또는 설치 경로)
    PATH 추가 %ORACLE_HOME%\bin
