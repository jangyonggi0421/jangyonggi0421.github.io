Linux / Unix 명령어

  • uname : 시스템의 이름과 버전, 커널 버전 등 시스템 정보를 확인할 때 사용
  • cat : 파일 내용 표시
  • chmod : 파일 권한 설정
  • chown : 소유자 변경
  • cp : 복사
  • exec : 새로운 프로세스 수행
  • find : 파일 찾기
  • fork : 새로운 프로세스 생성, 하위 프로세스 호출, 프로세스 복제
  • fsck : 파일 시스템을 검사하고 보수
  • ls : 현재 디렉터리 내의 파일 목록 확인
  • mount : 파일 시스템 마운트
  • unmount : 파일 시스템 마운트 해제
  • rm : 삭제
  • cd : 디렉토리 이동
