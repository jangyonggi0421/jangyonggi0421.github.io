Linux 원격 관리 설정 가이드

시스템 환경
  • OS: Rocky Linux 9.7 (Blue Onyx)
  • Kernel: Linux 5.14.0-611.36.1.el9_7.x86_64

방법 1. Cockpit 설치 및 설정

웹 브라우저를 통해 리눅스 서버를 직관적인 그래픽(GUI)으로 관리할 수 있게 해주는 오픈소스 도구입니다.

  • 주요 특징: 9090 포트 사용, 실시간 모니터링, 서비스 및 네트워크 제어
# 1. cockpit 설치
sudo dnf install -y cockpit

# 2. 부팅 시 자동 시작 설정 및 즉시 실행 
sudo systemctl enable --now cockpit.socket

# 3. 방화벽 허용 설정
sudo firewall-cmd --add-service=cockpit --permanent

# 4. 방화벽 규칙 적용
sudo firewall-cmd --reload

# 5. 접속 주소 (기본 포트: 9090)
# https://<내부 서버주소>:9090

방법 2. XRDP 설정

Windows의 원격 데스크톱 연결(RDP) 프로토콜을 사용하여 리눅스 GUI 환경에 접속합니다.

# 1. epel 저장소 추가
sudo dnf install epel-release -y

# 2. XRDP 설치
sudo dnf install xrdp -y

# 3. XRDP 자동 시작 설정 및 즉시 실행
sudo systemctl enable --now xrdp

# 4. 방화벽 열기 (기본 포트: 3389)
sudo firewall-cmd --add-port=3389/tcp --permanent

# 5. 방화벽 규칙 적용
sudo firewall-cmd --reload

# 6. 접속 정보
# 서버주소:포트번호 (예: 192.168.0.1:3389)
