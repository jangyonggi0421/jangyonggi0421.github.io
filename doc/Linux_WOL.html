Linux WOL 설정(Rocky Linux 9.7)

시스템 환경
  • OS: Rocky Linux 9.7 (Blue Onyx)
  • Kernel: Linux 5.14.0-611.36.1.el9_7.x86_64

1. BIOS/UEFI 설정

먼저 BIOS/UEFI에서 해당 옵션을 활성화

  • 컴퓨터 부팅 시 BIOS/UEFI 설정 화면으로 진입(보통 Del, F2, F10 키 중 하나를 누름)해서 각 메인보드 별 WOL 기능 활성화

    • 2. Rocky Linux 9.7 설정

    # 1. 네트워크 인터페이스 이름 확인
@localhost ~]$ nmcli connection show

# 예시 출력:
# NAME                UUID                                  TYPE      DEVICE
# 유선 연결 1          12345678-90ab-cdef-1234-567890abcdef  ethernet  enp0s5    

# 2. nmcli로 WOL 설정
@localhost ~]$ sudo nmcli modify "" 802-3-ethernet.wake-on-lan magic

# "" : 1번 결과의 NAME 항목 값 입력
# 예) sudo nmcli modify "유선 연결 1" 802-3-ethernet.wake-on-lan magic

# 3. 변경 사항 활성화
@localhost ~]$ sudo nmcli connection up ""

# 4. WOL 활성화 여부 확인
@localhost ~]$ nmcli connection show "" | grep "802-3-ethernet.wake-on-lan"
# 결과 예시: 802-3-ethernet.wake-on-lan: magic

# 또는
@localhost ~]$ sudo ethtool  | grep "Wake-on"
#  : 1번 결과의 DEVICE 항목 값 입력
# 예) sudo ethtool enp0s5 | grep "Wake-on"
# 결과 예시: Wake-on: g (g 일 경우, 활성화 상태)
