- windows 11 pro 24H2 26100.4946 기준
  1. 다운로드 URL : https://jdk.java.net/archive/
  2. OpenJDK-17.0.2_windows-x64_bin.zip 다운로드
  3. JAVA 폴더 생성
    • 예) 경로 : H:\dev\JAVA
  4. 파일 압축 해제
  5. 시스템 환경 변수 설정
    1. Windows 11 Pro 기준 : 설정 > 시스템 > 정보 > 고급 시스템 설정
    2. 환경 변수 클릭
    3. 시스템 변수 > 새로 만들기
      • JAVA_HOME 환경변수 생성
        • 변수 이름 : JAVA_HOME
        • 변수 값 : [디렉터리 찾아보기] JDK(JAVA) 설치 경로
      • Path 환경 변수 설정
        • 편집 > 새로 만들기
        • %JAVA_HOME%\bin 입력 후 최상단으로 이동(위로 이동) > 확인
      • CLASSPATH 환경 변수 생성 및 설정
        • 변수 이름 : CLASSPATH
        • 변수 값 : %JAVA_HOME%\lib
  6. 설치 확인 (명령 프롬프트/PowerShell) 
    java --version
